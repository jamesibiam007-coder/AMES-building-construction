## 🚀 **How to Use**

### **Quick Start (Local)**
1. Save all 3 files (index.html, styles.css, script.js) in the same folder
2. Open index.html in any web browser
3. Done! No server required

### **GitHub Pages (Free Hosting)**
1. Create a GitHub repository
2. Upload the 3 files
3. Enable GitHub Pages in Settings
4. Your site is live!

## 🎨 **Customization**

### **Change Colors**
Edit CSS variables in `styles.css` (line 8):
```css
--primary-color: #8B4789;      /* Main purple */
--dark-color: #2C1B3D;         /* Dark purple */
--accent-color: #E6B8E6;       /* Light purple */
```

### **Update Contact Info**
Find and replace in `index.html`:
- Phone numbers
- Email address
- Location

### **Change Images**
Replace image URLs with your own from:
- Unsplash
- Pexels
- Pixabay
